CatOS Игровая Консоль

Открытая прошивка портативной игровой консоли на базе ESP32 с OLED-дисплеем. Игры, утилиты и системные инструменты в одном устройстве!

CatOs

  • Микроконтроллер ESP32
  • OLED дисплей 128x64 (SPI)
  • 5 кнопок управления
  • Литий-ионный аккумулятор
  • Поддержка WiFi (STA и AP режимы)
Прошить ESP32

CatOS

  • Тетрис, Змейка, Flappy Bird
  • Ардуино дино, Понг, Арканоид
  • Системные настройки через веб
  • Файловый менеджер для LittleFS
  • Калькулятор, секундомер, таймер

CatOS Lite

  • Только режим читалки (шпоргалки)
  • 3 кнопки управления
  • I2C дисплей
  • Системные настройки через веб
  • Энергоэффективный
